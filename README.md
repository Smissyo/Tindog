# 🐶 TinDog

A fun, responsive, and modern landing page for a fictional Tinder-style dating application designed exclusively for dogs. This project showcases structured layouts and clean design implementation.

🔗 **Live Demo:** [View the live website here](https://Smissyo.github.io/Tindog/)

---

## 🚀 Features

* **Fully Responsive Design:** Adapts smoothly across mobile, tablet, and desktop viewports.
* **Modern Hero Section:** Clean call-to-action buttons for app downloads on both iOS and Android.
* **Feature Grid:** Highlights app benefits using custom inline SVG iconography.
* **Testimonials:** Clean layout featuring user reviews and social validation.
* **Interactive Pricing Cards:** Transparent tiered subscription plans for different user requirements.

---

## 🛠️ Built With

* **HTML5:** Semantic structuring of content elements.
* **CSS3:** Custom page layouts, color typography, and responsive media queries.
* **SVGs:** Scalable vector graphics used for clean, high-resolution custom icons.
